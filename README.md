# Loan Eligibility Engine

This project automates:
- Uploading user data
- Discovering loan products
- Matching users to eligible loans
- Sending notifications via SES

## Folder Structure

- **infra/** → AWS serverless deployment files  
- **n8n-workflows/** → JSON exports of n8n workflows  
- **backend/** → Lambda code  
- **docs/** → architecture diagrams & documentation
